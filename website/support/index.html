Hardware Required

  • Raspberry Pi 4
  • Raspberry Pi Touchscreen
  • Raspberry Pi Camera
    • Note: A cheaper camera can be used, however this camera is the minimum required to work at night.
  • OBD2 USB Cable
  • Arduino Uno
    • Note: Connecting wires and breadboards needed to connect the sensors are also required for a full 4 sensor array
  • HC-SRO4 Ultrasonic Distance Sensor (4 total)
  • 12v to 5v 5a Voltage Regulator
    • Used to run the Pi and its peripherals off the car battery
  • You will also need whatever cables are required to connect to your vehicle of choice's speaker system.
  • A 3D printer was used to create mounts and housings. File can be found HERE.

Installation into Vehicle

Installation will vary heavily from vehicle to vehicle. Hardware required to connect to the speakers in the car will also vary. Refer to your vehicle's make / model installation guides of After Market Headunits. The process will be relatively similar.

Links to clips of when we installed it in the 350z can be found HERE.

Installation of OpenZ on the Raspberry Pi

Install scripts can be found HERE.
Running "chmod +x install.sh" followed by "./install.sh" will automatically install everything onto the Raspberry Pi and configure the start up script to launch OpenZ Software on Boot.
