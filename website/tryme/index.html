While we at HnB Enterprises do not expect everyone to rush out and tear up their car to install this software, feel free to download the Jar file on our Github to play around with!
The main Jar containing the Music Player, OBD and GPS can be found HERE.
The Python OBD TCP server required for OBD to work properly can be found here HERE.

These features can be run off a laptop. In order to run the OBD, you will need an USB OBDII cable to plug into the car.
The Music Player is configured to watch the ~/Music/ directory of a GNU/Linux system as the Raspberry Pi runs Raspbian.
